  - Xuất file Excel (Export Excel)

### Filter Bar
- **Tabs**: Tất cả / Đang điều trị / Hoàn thành / Chưa phát sinh
- **Branch filter dropdown**: "Lọc chi nhánh tạo"
- **Search input**: "Tìm kiếm"
- **Filter button** (with badge count) + **Column settings button**

### Table
- **14 columns**: Checkbox, Họ và tên (avatar+code), Ngày sinh, Ngày hẹn gần nhất, Ngày hẹn sắp tới, Ngày điều trị gần nhất, Tình trạng điều trị (colored badge), Dự kiến thu, Công nợ, Tổng tiền điều trị, Tổng doanh thu, Thẻ thành viên, Nhãn khách hàng (tag button), Chi nhánh tạo, Thao tác

3. **Stop follow (link-off icon)** → Opens confirmation dialog:
   - Title: "Ngưng theo dõi khách hàng [Name]"
   - Message: "Bạn có chắc chắn muốn ngưng theo dõi khách hàng này?"
   - Buttons: Đóng, Xác nhận

## 9. COMMON UI PATTERNS

### Confirmation Dialogs
- Icon (colored circle with icon)
- Bold title
- Description text
- Warning text (if destructive, in red)
- Two buttons: Đóng (Close, outline) + Xác nhận (Confirm, primary/red)

### Modals
- Full-width header with title + X close button
- Content area with form fields
- Footer with action buttons (Đóng + Lưu)
- Close via X button, Đóng button, or Escape key

### Dropdowns
- Search input at top
- Scrollable option list
- Some have "+" add-new button
- Clear/remove buttons (X icons)

### Tables
- Fixed header row
- Sortable columns
- Checkbox selection
- Action buttons column
- Pagination: page numbers + rows-per-page selector + total count

### Empty States
- Cartoon character illustration
- "Không có dữ liệu" or "Chưa có [item] nào" text

### Color Labels (8 colors used in appointments)
1. Blue (default/selected with checkmark)
2. Red
3. Orange
4. Pink
5. Green
6. Teal
7. Light Blue
8. Purple

### Treatment Status Badges
- "Chưa phát sinh" (Not started) - gray
- "Đang điều trị" (In treatment) - blue/active
- "Hoàn thành" (Completed) - green
